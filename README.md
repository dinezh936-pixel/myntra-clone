# 🛍️ Myntra E-Commerce Frontend Clone

Welcome to my Myntra Clone repository! This project is a fully structural and styled frontend replica of the popular Myntra shopping website. 

I developed this project as part of my journey in learning Full-Stack Web Development and actively building in public to showcase my hands-on coding skills.

## 🚀 Tech Stack Used
* **HTML5:** Used for semantic structuring of the web pages, creating navigation bars, product grids, and footers.
* **CSS3:** Utilized for complete styling, layout management (Flexbox), hover effects, and ensuring a clean user interface that mirrors the original site.

## ✨ Key Features
* **Interactive Navigation:** Replicated the header section with search bars and categorized navigation links.
* **Product Grids:** Structured layout for displaying promotional banners and multiple clothing category sections.
* **Detailed Footer:** A comprehensive footer section containing links, app download badges, and copyright information.

## 🛠️ How to Run Locally
Since this is a pure frontend project built with HTML and CSS, running it is incredibly simple!
1. Clone this repository to your local machine.
2. Navigate to the project folder.
3. Double-click on the `index.html` file to open it in any modern web browser (Chrome, Firefox, Safari).
